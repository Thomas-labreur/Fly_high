## Description

Flheight est une GUI conçue intialement pour compter les mouches sur une image de tubes qu'elles remontent, et pour mesurer la hauteur de ces mouches.

## Lancer en ligne de commande
```bash
pip install -r requirements.txt
python flheight.py
```

## Générer un exe
```bash
pip install -r requirements.txt
pyinstaller --onefile --windowed --name Flheight flheight.py
```
